# Data Source

## Dataset Description
The project uses a publicly available customer churn dataset representing a telecom-like business scenario.

## Source
- Public dataset (e.g., Kaggle telecom churn dataset)
- Used strictly for educational and portfolio purposes

## Data Characteristics
- Customer tenure and service usage
- Billing and contract information
- Customer churn label

## Privacy & GDPR Considerations
- No personally identifiable information (PII) is included
- Customer identifiers are anonymized
- Dataset is suitable for GDPR-compliant analysis
