# Enterprise Customer Churn Analytics Platform (Azure)

## Overview
This project implements an end-to-end customer churn analytics platform using the Microsoft Azure ecosystem.
It is designed to reflect real-world data workflows commonly used in German enterprises.

## Business Problem
Customer churn directly impacts revenue and long-term growth. The goal of this project is to identify
customers at high risk of churn and provide actionable insights for retention strategies.

## Planned Architecture
- Local data processing (initial phase)
- Azure Data Lake Gen2 for storage
- Azure Synapse Serverless SQL for analytics
- Azure Machine Learning for churn prediction
- Power BI for business dashboards
